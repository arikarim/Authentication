# Authentication

> Rails project to demonstrate authentication using "Devise" gem

## Built With

- Ruby
- Rails
- Rubocop
- VSCode



## Getting Started

### Setup

- Open terminal
- Run "git clone https://github.com/arikarim/Authentication.git"
- cd into Authentication
- run "bundle install" to get all required gems
- run "rails server"
